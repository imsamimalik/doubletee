# doubleTee

## Features
  
- Login/Sign up User
  - Student
  - Faculty
  - Admin
- Add/Remove Courses
- Add/Remove Rooms
- Add/Remove Faculty
- Assign TimeTable

## Requiements

- Java (v11+)
- intelliJ Idea (preferable)
- MySQL
  - Install MySQL instance on your machine and update the credentials in src/main/resources/application.properties file
